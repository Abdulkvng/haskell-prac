# Haskell & Python Practice

A collection of practice exercises for learning Haskell and Python fundamentals.

## Haskell

- **practice.hs** - Basic Haskell functions (`map`, list operations)
- **test.hs** - Exercises for `map` with placeholder functions to fill in
- **wtest.hs** - Completed solutions for the `map` exercises

## Python

- **mer-training.py** - Recursive object flattening exercise: flatten nested dictionaries into dot-notation keys
- **recursion_unittest.py** - Unit tests for the `flatten_map` function

## Running

### Haskell
```bash
runhaskell practice.hs
runhaskell wtest.hs
```

### Python
```bash
python3 mer-training.py
python3 recursion_unittest.py
```
