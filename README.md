# 🚀 Haskell & Python Practice

A hands-on learning playground for sharpening functional programming skills in **Haskell** and mastering **recursion** in **Python**. This repo contains interactive exercises, solutions, and test suites designed to build a strong foundation in core programming concepts — from higher-order functions like `map` to recursive data structure traversal.

---

## 🟣 Haskell

Explore the power of functional programming through exercises focused on higher-order functions and list transformations.

| File | Description |
|------|-------------|
| 📄 `practice.hs` | Intro to Haskell — basic functions, `map`, and list operations |
| 🧩 `test.hs` | Fill-in-the-blank exercises: replace placeholders with the correct `map` functions |
| ✅ `wtest.hs` | Complete solutions to all `map` exercises for reference |

## 🐍 Python

Dive into recursion with a real-world coding challenge — flattening deeply nested dictionaries into clean, dot-notation keys.

| File | Description |
|------|-------------|
| 🏋️ `mer-training.py` | Recursive object flattening exercise with guided hints and a full test suite |
| 🧪 `recursion_unittest.py` | Standalone unit tests for the `flatten_map` function |

---

## ⚡ Getting Started

### 🟣 Run Haskell files
```bash
runhaskell practice.hs
runhaskell wtest.hs
```

### 🐍 Run Python files
```bash
python3 mer-training.py
python3 recursion_unittest.py
```

---

## 📚 What You'll Learn

- 🔗 **Higher-order functions** — using `map`, lambdas, and function composition in Haskell
- 🔄 **Recursion** — traversing and transforming nested data structures in Python
- 🧪 **Testing** — writing and running unit tests to verify your solutions
- 🧠 **Problem-solving** — building intuition for depth-first search and key-path construction
